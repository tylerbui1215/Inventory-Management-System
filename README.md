# Inventory Management System

## Overview
A full-stack inventory management web application built using
Java Spring Boot and Angular to manage products and inventory data.

## Features
- Full CRUD operations for inventory items
- Server-side validation for data integrity
- RESTful API architecture
- Automated database seeding on startup

## Tech Stack
- Backend: Java, Spring Boot
- Frontend: Angular
- Database: MySQL
- Tools: Git

## What I Learned
- Designing RESTful APIs
- Connecting frontend to backend services
- Data validation and error handling
- Full-stack application structure
